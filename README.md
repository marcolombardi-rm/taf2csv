taf2csv
==========
Semplice ma efficace applicazione scritta in Python 3.7 per la conversione in formato CSV della Tabella Attuale Fiduciali (TAF) fornita dall'AdE. Le coordinate scritte sul file .csv sono gia' invertite cosi' da poter essere utilizzate in ambiente CAD o GIS senza ulteriori manipolazioni.
